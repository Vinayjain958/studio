# **App Name**: Movie Scene Scripter

## Core Features:

- Mood/Theme Input: Text input field for entering the mood or genre of the scene.
- AI Script Generation: Generates scene titles, setting descriptions, character lists, dialogue, action descriptions, and camera angles based on user input.
- Professional Script Formatting: Displays the generated script in a structured screenplay format with character names, dialogue blocks, scene directions, and camera instructions.
- Generate Script Button: One-click button to trigger the script generation process.
- Regenerate Scene: Create alternate versions of the scene based on the same mood or theme.
- Dark/Light Mode Toggle: Allows users to switch between dark and light themes with saved preferences.
- Copy Script Button: Enables users to quickly copy the generated script to their clipboard.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a cinematic and mysterious feel.
- Background color: Very light lavender (#F0F8FF), almost white, to ensure readability.
- Accent color: Electric violet (#8F00FF) to highlight interactive elements.
- Body and headline font: 'Inter' for a modern, machined, objective, neutral feel.
- Responsive layout to ensure readability on various devices.
- Use minimalistic icons to represent different functions and actions.
- Subtle transitions and animations for generating scenes and switching between modes.
